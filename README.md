# Singularity Resources

This repository hosts the resource files (core binaries and rule sets) for the [Singularity Client](https://github.com/你的用户名/singularity-source).

The binaries distributed in the `Releases` section are repackaged versions of **Sing-box**, which is developed by **SagerNet**.
